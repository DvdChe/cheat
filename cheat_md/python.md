# Python Tricks

## Venv :

### Create Virtual Env
```
# Python3 :
python3 -m virtualenv --no-site-packages /location/of/env
# Default Python used by system : 
python -m virtualenv --no-site-packages /location/of/env
```

### Go to env : 
```
source /path/to/project/env_name_of_project/bin/activate
```
